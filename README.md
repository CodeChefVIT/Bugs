## Features
- Report Bugs in CC Project
- Allows Direct Comments into Github 
- Automated Issues Status
- Auto Generated Email For Issues 
- Auto Generated Labels For Issues

```
$ git clone https://github.com/CodeChefVIT/Bugs
$ cd Bugs/Backend 
$ npm install 
$ node app.js
```
